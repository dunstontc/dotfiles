# Vim #



## Leader Maps ##
```sh
<leader>
    │
    ├── b       # buffers...
    │   ├── l       # List Buffers
    │   ├── n       # Next Buffer
    │   ├── p       # Previous Buffer
    │   ├── q       # Force Close 
    │   ├── s       # Save all
    │   └── 
    │   
    ├── c       # config...
    │   ├── e       # edit...
    │   │   ├── f       # FTPlugin
    │   │   ├── v       # $MYVIMRC
    │   │   ├── p       # Plugin Settings
    │   │   └── 
    │   ├──
    │   └──   
    │  
    ├── h       # help...
    │   ├── i       # Open index.man
    │   ├── m       # Messages
    │   ├── d       # describe...
    │   │   ├── 
    │   │   └── 
    │   └── 
    │ 
    ├── t       # toggle...
    │   ├── f       # Folding
    │   ├── l       # Line Wrapping
    │   ├── m       # Table Mode
    │   ├── r       # Relative Numbers
    │   ├── W       # Invisible Characters
    │   └──  
    │ 
    ├── w       # window...
    │   ├── h       # Horizontal Split
    │   ├── v       # Vertical Split
    │   └──   
    │ 
    ├── q       # Quit 
    ├── r       # Redo
    ├── x       # Todos
    ├── `       # Terminal Split
    └──         # 

```
